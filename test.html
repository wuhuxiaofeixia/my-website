数学 中级 60分钟

1 第一部分：基础题

习题 1.1

基础

计算下面的表达式：

$$ \frac{3x^2 + 2x - 5}{x - 1} $$

其中 x = 3

查看提示

先将 x = 3 代入原表达式，然后进行计算。

习题 1.2

基础

解以下一元二次方程：

$$ 2x^2 - 5x + 3 = 0 $$
查看提示

可以使用求根公式：$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$，其中 a = 2, b = -5, c = 3

2 第二部分：进阶题

习题 2.1

进阶

证明以下等式：

$$ \sum_{i=1}^{n} i^3 = \left(\sum_{i=1}^{n} i\right)^2 $$
查看提示

可以使用数学归纳法证明，也可以利用等式 $\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$

3 第三部分：挑战题

习题 3.1

挑战

解决以下问题：

一个袋子里有5个红球和3个蓝球。小明随机取出2个球，然后又随机取出2个球。求取出的4个球中恰好有2个红球的概率。

查看提示

这是一个超几何分布问题。考虑所有可能的情况：第一次取(2红0蓝)、(1红1蓝)或(0红2蓝)，然后计算第二次取球相应的概率。
